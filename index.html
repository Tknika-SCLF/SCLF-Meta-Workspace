SCLF Meta-Workspace

Tknika-ko Smart Collaborative Learning Factory plataformako osagai guztiak zentralizatzen dituen ekosistema integrala.

🚀 Ikuspegi Orokorra

Gordailu honek SCLF ekosistemako modulu bakoitza era independentean kudeatzea ahalbidetzen du Git Submodules teknologia erabiliz. Horrela, proiektu bakoitzak bere bizitza propioa du, baina hub honetatik sistemaren ikuspegi global bat eta integrazio osoa mantentzen da.

  • 🐕 SCLF QuadrupedLau hanka dituen plataforma robotikoa.
